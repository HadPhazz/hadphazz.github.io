I acquired this DELL Optiplex 3010 SFF late 2020, I modded/customed it with an orange front panel, as well as upgrading some components.

The CPU is a Core i5 3470, 8Gb Dual-Channel RAM sticks, a 120Gb SSD for games, and a GT 710 from ASUS, from which I changed the passive cooler.

I can play Rocket League on medium quite well, as well as Minecraft, and it's perfect for videos. I'm planning to change the GPU to a Yeston RX550 Low Profile.

The cost of this computer is around 140€, which is a really great deal as I can still upgrade it for cheap !
